§1 Знакомство с Node

Nodejs - асинхронная, управляемая событиями JS платформа. 

1.1
1. Nodejs использует одонопоточную модель программирования. 2. Исполнительная среда Nodej не блокирует выполнение бизнес логики
во время ввода/вывода. Для этого в node используют три коцепции: события, асинхронные API, неблокирующий ввод/вывод. 3. Неблоки
рующий IO термин, означающий, что программа может обратится с запросом и затем занятся чем-то дркгим, когда операция будет заве
ршена, выполнится функция обратного вызова, которая оброботает результат.
